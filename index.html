ABOUT NEMO

> TOP

 

 

Nemo is a Grasshopper plugin for Ship Design.

 

Nemo enables automatic analysis of Ship Hull, single or multi, symmetrical or not. Nemo provides tools to import hull geometry, perform XZ symmetry, create stations - buttocks - waterlines curves, perform hydrostatics analysis, extract silhouettes curves, export cross-sections for structural analysis, perform resistance predictions, export meshes for potential flow simulations...

 

Please check the Facebook Page for latest news and updates

and the Grasshopper Group for questions or suggestions

MAIN FEATURES

Interoperability

Export ship data to other software, such as NavCad, Mars or GHS (coming soon)

Resistance

Estimate bare hull resistance, using Holtrop displacement or Savitsky planing methods

Design

Hull sectioning, Cross-section points extraction, NACA airfoil section generator

Hydrostatics

Compute hull hydrostatics and coefficients, Ship silhouette, Bulb and Transom analysis
